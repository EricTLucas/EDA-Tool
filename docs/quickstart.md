# Quickstart

This guide shows how to generate your first EDA report.

## 1. Prepare a dataset

Use any CSV file, for example:

# tests/sample_files/iris.csv

## 2. Run the CLI

```bash
eda tests/sample_files/iris.csv --out reports/
```

## 3. Open the report

reports/iris_report.html

You’ll see:

Summary tables

Distribution plots

Missing value analysis

Interactive tabs
